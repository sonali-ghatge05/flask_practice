# Flask Practice

A collection of Flask exercises covering fundamentals of backend web development.

## Topics Covered
- Routes and URL mapping
- Jinja2 templating
- Handling forms (GET/POST)
- MySQL database integration
- CRUD operations

Practiced as part of my backend development learning.

## MySQL Setup

1. Install MySQL locally or use a cloud instance.
2. Create a database: `CREATE DATABASE flask_practice;`
3. Update your database credentials in `connect_db.py`:
DB_HOST = "localhost"
DB_USER = "your_username"
DB_PASSWORD = "your_password"
DB_NAME = "flask_practice"
4. Install dependencies: `pip install flask mysql-connector-python`
5. Run the app: `python app.py`
